<h3>Dependencies</h3>


python 2.7, 
django 1.6, 
TwitterAPI 2.3, 
requests 2.7(ver. 2.7 required by TwitterAPI)

<h3>Instructions to execute</h3>
1. Inside the project root, execute $ **python manage.py syncdb**.
2. $ **python manage.py runserver**
3. Open **http://127.0.0.1:8000/** in web-browser.
